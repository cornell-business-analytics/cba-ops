# Content Model (High-Level)

## Page Schema (Conceptual)
Each page should include:
- `slug`
- `title`
- `summary`
- `seo` (title, description, social image)
- `sections` (ordered content blocks)
- `status` (draft | review | published)
- `updatedAt`

## Block Types (Conceptual)
- Hero block
- Rich text block
- Call-to-action block
- Team list block
- Project list block
- Event list block
- FAQ block
- Contact block

## Structured Data Collections
- Team members
- Projects
- Events

## Notes
- Exact schema definitions will be finalized after technology and CMS decisions.
